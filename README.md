# InnerCircle

InnerCircle is a real-time communication platform that allows users to join rooms and exchange messages instantly.  
The application supports live messaging, image sharing, typing indicators, and dynamic user presence.

## Live Demo
https://innercircle-yici.onrender.com

## Features

- Real-time messaging using WebSockets
- Room-based communication
- Image sharing with preview
- Typing indicators
- User avatars
- Message timestamps
- Dynamic online user list
- Modern responsive UI

## Tech Stack

- Node.js
- Express.js
- Socket.IO
- JavaScript
- Tailwind CSS
